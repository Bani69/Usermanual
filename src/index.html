Start a Project

Step 1: Enter Project Name

⦿ In the provided text input field, enter the name of your project. This should be a unique and descriptive name for the project you're working on. .

Step 2: Select a Plant Box

⦿ Use the first dropdown menu to select a plant box from the available options. A plant box is a specific container or area where you plan to plant and grow various plants. Choose the appropriate one for your project. .

Step 3: Select Project Configuration

In the next dropdown menu, choose the configuration for your gardening project. You have the following options:

⦿ Cole Crops: Ideal for growing vegetables like broccoli, cabbage, and cauliflower.

⦿ Ideal Environment for Cole Crops: Set up environmental conditions suitable for cole crop cultivation.

⦿ Environment for Fruits: Configure the environment for fruit-bearing plants like tomatoes, strawberries, or citrus trees.
